Lab: OpenTelemetry & Get Data In (GDI)

  less than a minute  

Goals

  1. Understand components involved for GDI, e.g.:
    1. OpenTelemetry Collector
    2. Prometheus exporters
  2. Understand how Developers, DevOps and SREs work with infrastructure and microservices
  3. Ability to ingest data from common stacks:
    1. Virtual Machines
    2. Containers and Container Orchestration (e.g. Docker, Docker Compose)
    3. Container Orchestration Platforms (e.g. Kubernetes)
  4. Understand troubleshooting 101 for components (commands)
